## Contribute
To add a new Linguistic Antipattern 
1. create a new mdx file in the ./content/antipatterns folder.

2. Follow the template found in ./content/template.md and fill out the mdx file.

3. Open a PR

## Quick start

### Prerequisites 

Node 16 LTS

### Setup
1. npm i
2. npm run dev
3. profit
